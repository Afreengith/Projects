# My Project Reports

This repository contains my academic and technical project reports from my Electronics and Communication Engineering coursework.

## Projects Included

- **Hand Gesture Controlled Wheelchair**  
  A wireless wheelchair controlled using hand gestures, built with Arduino Nano, MPU6050, nRF24L01, and L298 motor driver.

- **AGE AND GENDER PREDICTION IN FACE RECOGNITION SYSTEMS**  
  A detailed report on the internal architecture, working, and features of the 8051 microcontroller.

- **Digital Clock using 8051**  
  A major project implementing a digital clock using the 8051 microcontroller, including circuit design and code.

## Purpose
This repository showcases the documentation and work I have done during my undergraduate engineering projects.
